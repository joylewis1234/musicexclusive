# Music Exclusive

Music Exclusive is a React + Vite application backed by Supabase, Stripe, and Resend.

## Local Development

Requirements:

- Node.js
- npm

Start the app locally:

```sh
npm install
npm run dev
```

Build the production bundle:

```sh
npm run build
```

Run the Playwright end-to-end suite:

```sh
npm run test:e2e
```

## Core Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui
- Supabase
- Stripe
- Resend
